# EventGo-Website

EventGo is a simple front-end event booking website that allows users to explore events and access a login interface. The project focuses on clean UI design and user-friendly layout using HTML, CSS, and JavaScript.

## Features
- Home page with modern UI design
- Login and Sign Up interface
- Event browsing layout

## Technologies Used
- HTML
- CSS
- JavaScript

## Note
This project is a front-end demonstration only.
Backend functionality such as authentication and booking is not implemented.
